## Vectorised and Nested Switches

Fast vectorized and nested switches.

- [`iif()`](https://fastverse.org/kit/reference/iif.md) : Fast if else
- [`nif()`](https://fastverse.org/kit/reference/nif.md) : Nested if else
- [`vswitch()`](https://fastverse.org/kit/reference/vswitch.md)
  [`nswitch()`](https://fastverse.org/kit/reference/vswitch.md) :
  Vectorised switch

## Sorting

Parallel sort for strings and partial sort (N largest/smallest).

- [`psort()`](https://fastverse.org/kit/reference/psort.md) : Parallel
  Sort
- [`topn()`](https://fastverse.org/kit/reference/topn.md) : Top N values
  index

## Factors

Fast character to factor conversion and changing factor levels by
reference.

- [`charToFact()`](https://fastverse.org/kit/reference/charToFact.md) :
  Convert Character Vector to Factor
- [`setlevels()`](https://fastverse.org/kit/reference/setlevels.md) :
  Set levels of a factor object

## Unique Values and Counts

Fast duplicated and unique and count the number of times element(s)
occur.

- [`fduplicated()`](https://fastverse.org/kit/reference/funique.md)
  [`funique()`](https://fastverse.org/kit/reference/funique.md)
  [`uniqLen()`](https://fastverse.org/kit/reference/funique.md) : Fast
  duplicated and unique
- [`count()`](https://fastverse.org/kit/reference/count.md)
  [`countNA()`](https://fastverse.org/kit/reference/count.md)
  [`countOccur()`](https://fastverse.org/kit/reference/count.md) :
  count, countNA and countOccur

## Miscellaneous

Find a matrix position inside a larger matrix and share data between R
sessions.

- [`fpos()`](https://fastverse.org/kit/reference/fpos.md) : Find a
  matrix position inside a larger matrix
- [`shareData()`](https://fastverse.org/kit/reference/shareData.md)
  [`getData()`](https://fastverse.org/kit/reference/shareData.md)
  [`clearData()`](https://fastverse.org/kit/reference/shareData.md) :
  Share Data between R Sessions
